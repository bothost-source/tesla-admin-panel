# 🚗 Tesla Admin Panel

A modern responsive admin dashboard built using HTML, CSS, JavaScript and Bootstrap 5.

---

## Features

- 🔐 Admin Login
- 📊 Dashboard
- 🚗 Cars Management
- 👥 Customers Management
- 📦 Orders Management
- 💳 Payments
- 💬 Live Chat UI
- 📈 Analytics
- ⚙️ Settings
- 👑 Admin Management

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Bootstrap Icons

## How To Run

1. Download the project.

2. Open the project folder.

3. Start a local server.

Example:

```
python3 -m http.server 8000
```

4. Open:

```
http://localhost:8000
```
